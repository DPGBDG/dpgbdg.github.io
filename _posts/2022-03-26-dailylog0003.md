# 마크다운 사용법

마크다운에는 여러가지 사용법이 존재하며 Text editor 들이 존재한다
그중 유명한거로는 
Stack Edit 과 타이포라 라는 프로그램이 있다
스텍 에딧의 경우 온라인 프로그램
타이포라는 설치후 사용가능하다

나의 경우는 원툴 통일화가 편하여 vs code를 사용하였다
몇가지 플러그인이 필요하며
markdownlint
markdown all in one
korean spell checker vs code 
를 설치해주었다

드래그 한다음에 f1 누르고 korean spell check 를 찾으면 맞춤법 검사 해준다 ㅋ
